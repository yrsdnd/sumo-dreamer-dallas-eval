# Files in this folder

## Runtime — used by `run.py` / the trained model at every step
- `osm.sumocfg` — SUMO config the env loads on `reset()`
- `osm.net.xml.gz` — Dallas OSM road network (compressed)
- `osm.passenger.trips.xml` — 1,606 passenger vehicle trips
- `osm.motorcycle.trips.xml` — 1,350 motorcycle trips
- `osm.bus.trips.xml` — 454 bus trips + flows
- `output.add.xml` — edgeData statistics logger config

## Build-time / archival — used only if regenerating the network from OSM
- `City.netecfg` — netedit project config (for editing the .net.xml in netedit GUI)
- `osm.netccfg` — netconvert config (rebuild .net.xml from raw OSM)
- `osm_bbox.osm.xml.gz` — raw OpenStreetMap extract (input to netconvert)

The runtime files are the only ones the trained model interacts with. The
build-time files are kept in the repo so anyone could fully reproduce the
Dallas SUMO network from OpenStreetMap if needed.

`edgeData.xml` is intentionally NOT committed — it's autogenerated by SUMO on
each run.
